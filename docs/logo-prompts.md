# Signet Logo Generation Prompts

20 prompts for Gemini 3 Pro Image Preview (`gemini-3-pro-image-preview`).
Design language: industrial monochrome, blueprint schematic, sharp geometry,
registration marks, halftone texture. No color. No soft edges. No friendly SaaS.

Run with:
```bash
scripts/generate-image.sh "<prompt>" output/logo-XX.png
```

Pass existing outputs as reference images to maintain consistency across variants:
```bash
scripts/generate-image.sh "<prompt>" output/logo-XX.png output/logo-01.png
```

## Category A: Signet Ring / Seal Motif

A "signet" is a seal ring used to authenticate documents. Lean into that provenance.

### 01 — Signet Ring Impression
```
Logo design on pure black background. A single circular seal impression rendered
as a white wax stamp mark, showing the letter "S" pressed into it in a bold
monospace typeface. The seal has hairline registration marks at the cardinal
points like a technical crosshair overlay. Cracked and textured like an actual
wax impression but rendered in strict black and white only. No color. Halftone
grain texture across the surface. Stark, industrial, brutalist graphic design.
```

### 02 — Embossed Seal Matrix
```
Logo design on pure black background. A metallic signet ring matrix viewed from
directly above, showing a reversed monospace "S" carved into the flat face of
the ring. The metal surface is rendered with fine engraved crosshatch lines like
an etching plate. Tiny L-bracket registration marks at the four corners of the
design. Monochrome, no color, white on black. Technical illustration style like
a patent drawing or engineering schematic. High contrast.
```

### 03 — Broken Seal
```
Logo design on pure black background. A circular wax seal that has been broken
in half, with the two halves slightly separated. The seal bears an abstract
geometric "S" mark. Fine cracks radiate from the break. Rendered entirely in
white and light gray on black, with halftone dot texture like risograph printing.
Below the seal, the word "SIGNET" in a wide-tracked uppercase geometric sans-serif
typeface. No color. Stark and dramatic.
```

## Category B: Circuit / Node Network

Signet is infrastructure — memory nodes, graph connections, persistent data paths.

### 04 — Memory Node Graph
```
Logo design on pure black background. An abstract network graph forming the
shape of the letter "S". Nodes are small circles with crosshair marks through
them. Connections between nodes are thin straight lines with right-angle bends
like circuit board traces. Some nodes glow slightly brighter than others suggesting
active memory states. Entirely monochrome white on black. Below the mark, "SIGNET"
in uppercase monospace type with wide letter-spacing. Technical schematic aesthetic.
```

### 05 — Circuit Trace Letterform

```
Logo design on pure black background. The word "SIGNET" rendered as if each

suggesting data flow. Small component markers (resistor symbols, capacitor marks)
appear as decorative elements along the traces. Monochrome, no color. Clean
technical illustration. Registration marks at corners of the composition.
```

### 06 — Constellation S
```
Logo design on pure black background. An abstract letterform "S" made from a
constellation of dots connected by thin lines, like a star map or astronomical
chart. Each dot is a different size suggesting hierarchy — larger nodes are
primary memories, smaller ones are connections. Faint grid lines in the background
like graph paper. The word "SIGNET" below in uppercase geometric sans-serif with
wide tracking. Monochrome white on black. No color. Precise and mathematical.
```

## Category C: Geometric Abstraction

Pure form. No literal references. The mark carries meaning through shape alone.

### 07 — Concentric Target
```
Logo design on pure black background. Three concentric incomplete circles of
different stroke weights, offset slightly from center to create tension. A single
crosshair intersects the center point. The overall composition suggests a target
or calibration mark. Hairline registration ticks at 0, 90, 180, and 270 degrees.
Below, "SIGNET" in uppercase condensed geometric sans-serif. Entirely white on
black. No fills, strokes only. Technical, precise, like an oscilloscope display.
```

### 08 — Hexagonal Lattice
```
Logo design on pure black background. A single hexagon constructed from six
equilateral triangles, with a smaller hexagon nested inside rotated 30 degrees.
The intersections create a geometric pattern suggesting crystalline structure or
molecular bonds. Fine crosshair marks at each vertex. Below, "SIGNET" in wide-
tracked uppercase monospace. Monochrome white on black. No color, no gradients.
Sharp precise lines like an engineering blueprint. Halftone grain overlay.
```

### 09 — Split Diamond
```
Logo design on pure black background. A diamond shape (square rotated 45 degrees)
split vertically down the center with a thin gap. The left half is solid white,
the right half is rendered in fine horizontal scan lines creating a halftone
effect. The split suggests duality — stored and recalled, written and read.
Below, "SIGNET" in bold uppercase geometric sans-serif with tight tracking. No
color. High contrast black and white only.
```

### 10 — Nested Brackets
```
Logo design on pure black background. The letter "S" formed entirely from nested
square brackets and L-shaped corner marks — the same registration marks used in
print production. Multiple layers of brackets at different scales create the
curves of the S through angular approximation. Below, "SIGNET" in monospace type.
Pure white on black. The aesthetic of a terminal output or code editor. Precise
pixel-level geometry.
```

## Category D: Typographic / Wordmark

The name itself as the logo. Type as design.

### 11 — Stencil Cut

```
as if spray-painted through the stencil onto a dark surface. A thin horizontal
rule above and below the word with small crosshair marks at each end. White on
```

### 12 — Blueprint Annotation
```
Logo design on dark navy-black background. The word "SIGNET" rendered as if it
were a technical blueprint annotation — thin uppercase letters in a drafting
typeface with dimension lines extending from the top and bottom of the text
showing measurements. Small triangular arrowheads at the ends of dimension lines.
A revision number "v0.63" in smaller type at bottom right. Grid lines faintly
visible in background. Monochrome — white and light gray on near-black.
```

### 13 — Monospace Stack
```
Logo design on pure black background. The word "SIGNET" with each letter stacked
vertically in bold monospace type, aligned left. To the right of each letter, a
thin horizontal line extends to the edge of the composition like a label callout
on a technical diagram. Each line terminates with a small tick mark. The overall
shape is tall and narrow like a vertical sidebar. White on black. Clean, minimal,
systematic.
```

### 14 — Overprint Offset
```
Logo design on pure black background. The word "SIGNET" printed twice — once in
```

## Category E: Symbolic / Conceptual

### 15 — Fingerprint Circuit
```
Logo design on pure black background. A partial fingerprint where the ridge

transition zone is where biology meets technology. Below, "SIGNET" in uppercase
geometric sans-serif. Entirely monochrome white on black. No color. The
fingerprint represents identity; the circuits represent persistence. Fine
detailed linework.
```

### 16 — Memory Address Block
```
Logo design on pure black background. A rectangular block resembling a hex memory
dump display: rows of monospace hex characters (0xAF2A, 0xBE01, etc.) with the
letters S-I-G-N-E-T highlighted brighter than the surrounding data, scattered
across different positions in the dump as if they're memory addresses being read.
A thin border with corner registration marks. Monochrome. The aesthetic of a
debugger or memory inspector tool.
```

### 17 — Persistence Loop
```
Logo design on pure black background. An infinity symbol (lemniscate) rendered
with a single continuous line that has a subtle break at the crossover point,
with small arrowheads suggesting continuous flow direction. At the center
crossover, a small diamond node mark. The line has varying weight — thicker at
the outer curves, thinner at the center. Below, "SIGNET" in condensed uppercase
sans-serif. White on black, no color. The infinity represents persistent memory
across sessions. Clean vector aesthetic.
```

### 18 — Key Impression
```
Logo design on pure black background. A skeleton key viewed from above (the bit
end facing the viewer) showing the ward pattern — the unique cuts that make each
key individual. The ward pattern subtly forms an abstract "S" shape. The key bow

surfaces. Below, "SIGNET" in serif capitals with wide tracking. Elegant but stark.
```

## Usage Notes

- **Output directory**: create `output/` in the project root or wherever convenient
- **Iteration**: pass your best output as a reference image to the next prompt for consistency
- **Progressive refinement**: generate variants by modifying one element at a time
- **Combination**: mix elements from different categories (e.g., the fingerprint-circuit
  motif as a seal impression, the constellation as a ring matrix)
- **Text rendering**: Gemini 3 handles text well but may need multiple attempts for
  perfect letterforms. If "SIGNET" renders poorly, try the mark without text and
  composite the wordmark separately
- **Aspect ratio**: these prompts assume roughly square output. For horizontal lockups
  add "wide horizontal composition, 16:9 aspect ratio" to the prompt
